## Now you can!

Just download this repository!

### Steps

1. Clone this repository
2. Duplicate `config.py.example` and rename it to `config.py`
3. Import requirements (this can be done automatically with `pip install -r "requirements.txt"`)
4. Run `akivagraph.py`
